### カラム定義

| カラム | 説明 | 取り得る値 |
|--------|------|----------|
| SCR-ID | 画面の一意識別子（3桁ゼロ埋め連番） | `SCR-001`, `SCR-002`, ... |
| 画面名 | 画面の日本語名称 | 自由記述（例: ユーザー一覧画面） |
| 種別 | 画面のレイアウト種別 | `一覧` / `詳細` / `フォーム` / `ダッシュボード` / `モーダル` / `ウィザード` / `エラー` / `未分類` |
| 優先度 | 実装の優先度 | `MUST`（必須）/ `SHOULD`（推奨）/ `COULD`（あれば良い） |
| MVPスコープ | MVP（最小構成）に含むか | `in`（含む）/ `out`（含まない） |
| 担当 Epic | この画面を実装する Epic | `ES-NNN`（Epic 仕様書の番号） / `未割当` |
| 元 AC 参照 | この画面を導出した AC の ID | `AC-ENNN-NN`（複数の場合はカンマ区切り）/ `手動追加`（AC以外から追加した場合）/ `ドメインモデル`（集約から補完した場合） |
