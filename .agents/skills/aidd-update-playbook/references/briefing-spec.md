# /aidd-update-playbook ブリーフィング仕様

各ステップ完了時に、`aidd-framework/references/briefing-format.md` の 6 ステップに従ってブリーフィングを実行する。

## ステップ別のブリーフィング内容

### Step 2 完了時（影響セクション判定）

| 要素 | 必須提示内容 |
|------|------------|
| 全体概要 | 影響セクション数 / 影響なしセクション数 |
| 判断ポイント | 間接的影響の判定根拠（直接的影響は自明なため省略可） |
| AI 補完 | AI が「影響あり」と判定した間接的影響のセクション |
| 自信度 | 間接的影響の判定で自信度が低い箇所 |
| 意思決定 | 影響セクションの追加・除外の判断を求める |

### Step 3 完了時（再執筆）

| 要素 | 必須提示内容 |
|------|------------|
| 全体概要 | 更新したセクション数 / 変更箇所数 |
| 判断ポイント | 固有価値を維持するために行った判断（何を残し何を変えたか） |
| AI 補完 | AI が追加した記述（つまずきポイント、対話例等） |
| 自信度 | 再執筆内容で自信度が低い箇所 |
| 意思決定 | 更新案の承認判断を求める |

### Step 4 完了時（リンク検証）

| 要素 | 必須提示内容 |
|------|------------|
| 全体概要 | 検証リンク数 / リンク切れ数 |
| 判断ポイント | リンク切れの修正案の根拠 |
| 意思決定 | リンク切れ修正の承認判断を求める（リンク切れがある場合のみ） |
