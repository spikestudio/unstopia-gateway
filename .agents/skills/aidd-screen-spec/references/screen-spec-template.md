## フィールド定義

| フィールド名 | 型 | 必須/任意 | 説明 |
|------------|-----|----------|------|
| ユーザー名 | 文字列 | 必須 | 表示名（最大50文字） |
| メールアドレス | 文字列 | 必須 | ログインID |
| ロール | 選択肢 | 必須 | 管理者 / 一般ユーザー |
| 作成日 | 日付 | 任意 | ISO 8601 形式 |

## バリデーションルール

| フィールド名 | ルール種別 | ルール内容 |
|------------|----------|----------|

| ロール | 選択肢 | 管理者 / 一般ユーザー のいずれか |

## 状態設計

### 空状態

| 項目 | 内容 |
|------|------|
| 発生条件 | ユーザーが1件も登録されていない場合 |
| 表示内容 | 「ユーザーがいません」メッセージ + 「新規作成」ボタン |
| アクション | 「新規作成」ボタンで SCR-002（ユーザー作成画面）へ遷移 |

### ローディング状態

| 項目 | 内容 |
|------|------|
| 発生条件 | ユーザー一覧 API リクエスト中 |
| 表示内容 | スケルトンスクリーン（テーブル行のプレースホルダー） |
| タイムアウト | 30秒でエラー状態に遷移 |

### エラー状態

| 項目 | 内容 |
|------|------|
| 発生条件 | GET /api/v1/users が 5xx エラー / ネットワーク切断 |
| 表示内容 | 「データの読み込みに失敗しました」+ 「再試行」ボタン |
| アクション | 「再試行」ボタンで API を再リクエスト |
