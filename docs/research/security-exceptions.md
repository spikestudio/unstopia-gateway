### 1. protobufjs@6.8.8 [CRITICAL]

| 項目 | 内容 |
|------|------|
| CVE | CVE-2026-41242 |
| 脆弱性 | Arbitrary code execution |
| 修正バージョン | >= 7.5.5 |
| 経路 | `@whiskeysockets/baileys@7.0.0-rc.9` → `@whiskeysockets/libsignal-node@2.0.1` → `protobufjs@6.8.8` |
| 修正不可理由 | baileys は 7.0.0-rc.9 (RC段階)。libsignal-node が 6.x に依存しており、6.x には修正バージョンが存在しない |
