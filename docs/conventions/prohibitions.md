## コード

- `aidd-framework/` への直接変更禁止（フレームワーク更新で上書きされる）
- jinn (upstream) のパブリック API の破壊的変更禁止（アップストリームマージ容易性を保つ）
- 外部ライブラリの無確認追加禁止
- `@ts-ignore` / `eslint-disable` の suppress 無断使用禁止

## プロセス

- Issue なしの作業開始禁止
- Issue 参照なしの PR 作成禁止
- main ブランチへの直接コミット禁止
- Epic Issue 作成時の Milestone 未紐付け禁止（`--milestone` フラグは必須）（#172）
- ファイル分割 Epic 完了後、分割先の全ファイルのサイズ未確認禁止（500行以下であることを確認）（#171）
