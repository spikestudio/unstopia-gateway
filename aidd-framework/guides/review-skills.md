# レビュー体制: 3 層構造

## 概要

レビューは 3 つの層で段階的に品質を担保する。各層はプロセスに組み込まれており、独立したレビュースキルを個別に呼び出す必要はない。

## 3 層のレビュー構造

| 層 | スキル | ゲート | 責務 |
|----|--------|--------|------|
| **Task レベル** | `/aidd-impl` selfcheck | — | コミット単位のセルフレビュー（規約準拠・AC 準拠・テスト通過） |
| **Epic レベル** | `/aidd-epic-review` | G5 | Epic 全体の 3 観点チェック（AC 充足・設計整合性・コード品質）+ gate 制御 |
| **Phase レベル** | `/aidd-phase-review` | G6 | Phase 完了検証（成功基準達成・マスタドキュメント最新化） |

## 各層の詳細

### Task レベル: `/aidd-impl` selfcheck

`/aidd-impl` が各 Task の実装完了時にセルフレビューを実行する。問題があればその場で修正し、修正済みの状態でコミットする。

- 規約違反の検出と修正
- AC 準拠の確認
- テスト通過の確認
- autofix 相当の自動修正はこのループ内で完結

### Epic レベル: `/aidd-epic-review`

Epic の全 Task 完了後、PR マージ前に実行する統合レビュー。

- **3 観点チェック:** AC 充足・設計整合性・コード品質
- **Gate 制御:** `gate:reviewed` ラベルの付与
- **ブリーフィング:** レビュー結果を人間に報告し、マージ承認を得る

### Phase レベル: `/aidd-phase-review`

Phase の全 Epic 完了後に実行する Phase 完了検証。

- 成功基準の達成状況確認
- マスタドキュメントの最新化確認
- G6 マイルストーン通過準備

## 廃止されたスキル

以下のスキルは廃止され、上記 3 層構造に吸収された。

| 旧スキル | 吸収先 | 理由 |
|---------|--------|------|
| `/aidd-pr-review` | `/aidd-epic-review` | PR レビュー機能は Epic レビューに統合 |
| `/aidd-review` | 各スキルのセルフレビューループ | レビューエンジンは各スキルに内蔵 |
| `/aidd-autofix` | 各スキルのセルフレビューループ | 自動修正は各スキルに内蔵 |
| `/aidd-wrapup` | `/aidd-epic-review` | Epic 完了処理は Epic レビューに吸収 |
| `/aidd-brief` | 各スキルのブリーフィングステップ | ブリーフィングは各スキルに内蔵 |
