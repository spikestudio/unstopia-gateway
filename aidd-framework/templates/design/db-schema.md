## セルフチェック（G3 対応）

- [ ] 全テーブルのカラムに DB 固有型が指定されている

- [ ] 外部キーの ON DELETE / ON UPDATE 動作が全て定義されている
- [ ] インデックスに作成理由が記載されている
- [ ] ドメインモデルの全属性が DB カラムにマッピングされている
- [ ] マイグレーションファイル（DDL）をこの定義から生成できるレベルの詳細さがある
- [ ] 「AI が迷うポイント」の全項目にプロジェクトの方針が記入されている
- [ ] ADR の決定事項と矛盾していない
